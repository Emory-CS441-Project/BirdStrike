# BirdStrike Analysis

Data driven article exploring bird strike dataset. View online [here](https://emory-cs441-project.github.io/BirdStrike/).

## Running Locally
Code expects Node version 22 and npm version 11. Higher versions should work but not gaurenteed.

1. Clone the repo `git clone https://github.com/Emory-CS441-Project/BirdStrike.git`
2. Install dependances `npm i`
3. Run in dev mode `npm run dev`
4. View on local host link should be in terminal if not default is `http://localhost:5173/`
