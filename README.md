# BirdStrike Analysis

Data driven article exploring bird strike dataset. View online [here](https://emory-cs441-project.github.io/BirdStrike/).

## Running Locally
Code expects Node version 22 and npm version 11. Higher versions should work but not guaranteed.

1. Clone the repo `git clone https://github.com/Emory-CS441-Project/BirdStrike.git`
2. Install dependencies `npm i`
3. Run in dev mode `npm run dev`
4. View on local host link should be in terminal if not default is `http://localhost:5173/`

## AI Disclosure
Our approach to AI was as follows: we came up with our ideas for how the visualization would look, coded up working versions ourselves, and asked for AI assistance on d3 documentation parts we needed help with. This was useful for things like making sure the graphs actually look smooth and appealing and making sure we had no bugs in our map, which has several moving parts in the background. All the ideas for the visualizations themselves, from inception to fully fleshed out, were are own, and we used AI to polish the final product.
